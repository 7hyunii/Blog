React 18에서 추가된 주요 기능들을 살펴보고, 실전에서 어떻게 활용할 수 있는지 알아봅니다.

## Concurrent Rendering

React 18의 가장 큰 변화는 Concurrent Rendering입니다. 이를 통해 React가 여러 버전의 UI를 동시에 준비할 수 있게 되었습니다.

```javascript
import { useTransition } from 'react';

function App() {
  const [isPending, startTransition] = useTransition();
  
  const handleClick = () => {
    startTransition(() => {
      // 우선순위가 낮은 업데이트
      setTab('posts');
    });
  };
}
```

## Automatic Batching

이전 버전에서는 React 이벤트 핸들러 내부에서만 배치 처리가 가능했지만, React 18에서는 Promise, setTimeout, 네이티브 이벤트 핸들러 등 모든 곳에서 자동으로 배치됩니다.
