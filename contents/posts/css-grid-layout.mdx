CSS Grid를 활용하여 깔끔하고 유연한 반응형 레이아웃을 만드는 방법을 알아봅니다.

## Grid 기본 개념

CSS Grid는 2차원 레이아웃 시스템입니다. 행과 열을 동시에 다룰 수 있어 복잡한 레이아웃도 쉽게 구현할 수 있습니다.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}
```

## 자동으로 크기 조절되는 그리드

`auto-fit`과 `minmax`를 함께 사용하면 반응형 그리드를 쉽게 만들 수 있습니다.

```css
.responsive-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));
  gap: 1rem;
}
```

이 방법을 사용하면 미디어 쿼리 없이도 반응형 레이아웃을 구현할 수 있습니다.
